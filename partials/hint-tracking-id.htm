How to get the Measurement ID and Stream ID

  1. Login to: Google Analytics

  2. Click on the admin button in the bottom left of the screen.

  3. Create a new a new account clicking on the 'Create Account' button.

  4. Account > Property > Data Streams.

  5. Click on 'Web' and then on the 'Add Stream' button (if there isn't a data stream created yet).

  6. Make sure the 'Enhanced measurement' checkbox is turned on to see all measuring options turned on.

  7. Copy the 'MEASUREMENT ID' it starts with a 'G-' and also the 'STREAM ID' which should be a number.
